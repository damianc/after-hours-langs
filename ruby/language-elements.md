## Numbers

| Symbol | Example |
|--------|---------|
| + | 5 + 2 => 7  5 + 2.0 => 7.0 |
| - | 5 - 2 => 3 |
| * | 5 * 2 => 10 |
| / | 5 / 2 => 2  5 / 2.0 => 2.5 |
| % | 5 % 2 => 1 |
| ** | 5 ** 2 => 25 |

| Function | Example |
|----------|---------|
| abs | -3.abs => 3  3.abs => 3 |
| div | 10.div(2) => 5 |
| zero? | 10.zero? => false |
| ceil | 2.6.ceil => 3 |
| floor | 2.6.floor => 2 |
| round | 2.635.round(2) => 2.64 |
